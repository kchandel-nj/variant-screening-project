# Man Spine (`public/spine/man/`)

Keep this folder **complete in git / submissions** — it is the canonical man bundle for the take-home.

- **`animations.json`** / **`ANIMATIONS.md`** — animation names (optional refresh: `npm run sync-spine-catalogs`).
- **`skeleton.json`**, **`skeleton.atlas`**, **`skeleton*.png`** — runtime assets (maintainers: restore via **`npm run copy-spine`** from the monorepo if needed).

See [SETUP.md](../../SETUP.md) in the take-home root.
