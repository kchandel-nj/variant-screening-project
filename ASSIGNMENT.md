# Variant: Gameplay Programming Take-Home Assignment

## The Challenge

At Variant, we believe play is the new language of connection. We want to see how you turn a simple asset into a compelling, "remixable" experience. Your goal is to build a 60-second gameplay loop using **Phaser 4** and a provided **Spine** character.

## The Task

We have provided a "Hello World" style starter project in this folder. It contains:

- A basic Phaser setup (**Phaser 4.0.0-rc.6** — see `package.json`).
- Variant **shared man** Spine rig under `public/spine/man/` (`animations.json` + `ANIMATIONS.md` list every animation). The sample scene uses **`Walk`** / **`Jump`**.
- A simple script that makes the **man** character walk across the screen (`Walk` / `Jump`).

**Your mission:** Take this minimal foundation and build a playable **micro-game** or interactive scene.

## Requirements

- **The Hero:** Your game must center around the provided Spine character. Use its animations to give the game life.
- **The Loop:** Create at least one solid gameplay loop (e.g. a "meme-able" physics interaction, a rhythmic mini-game, or a creative control scheme).
- **Duration:** Roughly one minute of engagement.
- **Tech stack:** JavaScript within the provided Phaser framework (**Phaser 4.0.0-rc.6** in this starter). The canvas is **9:16 portrait** (720×1280) to match Variant’s mobile games.
- **The "Variant" factor:** We value creative logic over polish — a broken but original idea beats a generic clone.

## What we’re evaluating

- **Code literacy** — Phaser/Spine integration; readable logic for teammates.
- **Mechanical feel** — Controls and motion (snappy, floaty, intentional).
- **Problem solving** — Constraints and Spine/workarounds.
- **Terminal proficiency** — Easy to run: see **[CANDIDATE_INSTRUCTIONS.md](./CANDIDATE_INSTRUCTIONS.md)** (`npm install` + `npm start`).

## Submission

- **Code:** GitHub repo link or ZIP (exclude `node_modules`).
- **Dev log (short):** README section or file — what you built, one technical challenge, one thing you’d add in 48 hours.

Submit by the date/time given in your recruiting email (e.g. **Thursday, March 26th at 5pm ET** to grace@tryvariant.com unless instructed otherwise).
